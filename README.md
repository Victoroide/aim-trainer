# Aim Trainer

## Introduction
Welcome to Aim Trainer, a simple yet engaging Python game designed to offer a fun break from your daily grind. Whether you're taking a breather between university assignments or just need a quick distraction, this game is perfect for a short, refreshing session to sharpen your aiming skills and relax your mind.

## Features
- **Quick Gameplay**: Jump right into the game with no complex setup required.
- **Dynamic Targets**: Targets appear at random locations, offering a new challenge every time.
- **Score Tracking**: Keep track of your accuracy and improvement with an in-game scoring system.
- **Customizable Settings**: Tailor the game to your screen size and preferences.

## Technologies Used
- **Python**: The core programming language of the project.
- **Pygame**: A powerful library for creating games in Python.

## Getting Started
To get started with Aim Trainer, follow these steps:
1. Ensure you have Python installed on your machine.
2. Clone the repository: `git clone https://github.com/Victoroide/aim-trainer.git`
3. Install Pygame: `pip install pygame`
4. Run `app.py` to start the game.

## How to Play
- The game starts immediately upon launch.
- Click on the targets as they appear on the screen.
- Try to hit as many targets as possible within the time limit.
- Your score will be displayed at the end of each round.
